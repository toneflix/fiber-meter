# FiberMeter JavaScript SDK

Server-side JavaScript SDK for FiberMeter usage metering, prepaid customer
balances, payment requests, structured API errors, and signed webhooks.

See the [complete SDK guide](../../docs/05-sdk.md) for installation,
authentication, every method, response contracts, idempotency, error handling,
and secure webhook verification.

Build and test this workspace package with:

```sh
pnpm --filter @fibermeter/sdk build
pnpm --filter @fibermeter/sdk test
```
